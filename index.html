Pooty.js: A VMC framework for Javascript

that doesn't pretend to be anything it isn't.

Let's clean up that view.

You can use Pooty on an existing webpage without modifying the HTML. For advanced capabilities, you only need to learn three new tags: <poothtml>, <bucket>, and <template>. They do exactly what they sound like.
<html>
<body>
<span class="message">Helloooooooo Tina!</span>
</body>
</html>
/* A Pooty-compatible page! Pooty can change the message inside the span, reactively, without any special HTML syntax. */
